# HBnB Technical Documentation

## What is this about?

The entire project covers:

* **High-Level Architecture:** The structural design of an application using a layered approach.
* **Business Logic Layer:** The core entities, their attributes, and relationships.
* **API Interaction Flow:** The sequence of operations for handling client requests.

### Overview

HBnB is a web-based application that facilitates the booking of listings (rooms or houses) by users. The system is designed to be modular, scalable, and easy to maintain, utilizing a clear separation of concerns between the presentation, business logic, and persistence layers.
